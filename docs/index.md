# Polygon Crusher 9000

**Fast, clean mesh decimation for Blender — without destroying your normals.**

Polygon Crusher 9000 is a Blender addon that reduces mesh polygon count while transferring the original surface normals back onto the result — so decimated meshes keep their smooth shading even at aggressive reduction levels.

---

## Key Features

- **Smart decimation** — high-quality mesh reduction that preserves shape and silhouette
- **Normal preservation** — keeps smooth shading intact after decimation
- **UV transfer** — optionally re-projects the original UV map onto the crushed result
- **Lossless mode** — removes only redundant geometry with zero visual change; ideal for cleaning up over-subdivided or imported hard-surface models
- **Live progress bar** — Blender stays responsive during long crushes; press **ESC** to abort cleanly
- **Session stats** — per-object triangle counts and timing shown after each crush
